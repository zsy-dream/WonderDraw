# 🚀 零基础用户启动攻略

## 1️⃣ 一键启动（推荐）

双击运行文件：
```
一键启动.bat
```

即可自动打开浏览器访问项目！

---

## 2️⃣ 手动启动

如果一键启动失败，按以下步骤操作：

### 第1步：安装 Node.js（如果未安装）

1. 下载地址：https://nodejs.org/zh-cn/
2. 选择 LTS 版本下载
3. 双击安装，一直点击"下一步"

验证安装：
- 打开"命令提示符"（按 Win+R，输入 cmd，回车）
- 输入：`node -v`
- 如果显示版本号（如 v18.x.x），说明安装成功

---

### 第2步：启动前端（必须要）

1. 打开项目文件夹
2. 右键点击空白处 → "在此处打开命令窗口"
3. 依次输入：

```bash
cd frontend
npm install
npm run dev
```

4. 看到类似 `> Local: http://localhost:5173/` 提示后，打开浏览器访问：
   ```
   http://localhost:5173
   ```

---

### 第3步：启动后端（可选，如需真实AI功能）

1. 新开一个命令窗口（保持前端窗口运行）
2. 依次输入：

```bash
cd backend
python app_simple.py
```

3. 看到提示 `Running on http://localhost:5001` 表示成功

---

## 3️⃣ 常见问题

| 问题 | 解决方案 |
|------|----------|
| 双击 bat 没反应 | 右键 → "以管理员身份运行" |
| 提示"npm 不是命令" | 先安装 Node.js（上方第1步） |
| 提示"找不到目录" | 确认在正确的项目文件夹打开命令窗口 |
| 前端白屏 | 按 F12 打开控制台，截图发给我看 |
| 关闭命令窗口就关了 | 后台需要一直开着 |
