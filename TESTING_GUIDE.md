# 功能测试清单

## 项目启动步骤

### 1. 启动后端
```bash
cd backend
python app_simple.py
```
后端启动成功后会显示：`Running on http://0.0.0.0:5001`

### 2. 启动前端
```bash
cd frontend
npm run dev
```
前端启动成功后会显示：`Local: http://localhost:5173`

## 常见问题

### 后端启动失败
检查Python依赖：
```bash
pip install -r requirements.txt
```

### 前端启动失败
检查Node依赖：
```bash
cd frontend
npm install
```

### 端口冲突
后端默认5001，前端默认5173。如被占用：
- 后端：`python app_simple.py` 会自动尝试其他端口
- 前端：修改 `vite.config.js` 中的port
